# ENSTICP-PFE-Abdelmouaiz
# 🎓 PFE: Zero-Touch Deployment for IoT Edge Nodes

This repository contains the LaTeX source code for my final-year engineering project (Projet de Fin d'Études) to obtain the degree of **Ingénieur d'État en Télécommunications** at ENSTICP.

##  Project Overview
This project focuses on the architectural planning and practical implementation of **Zero-Touch Deployment and Cloud-Native Orchestration of Edge Nodes for IoT**. The proof of concept involves overcoming legacy hardware constraints using Virtual Machines and leveraging tools like K3s and GitOps methodologies.

##  Automated PDF Compilation
This repository is configured with **GitHub Actions**. Every time a commit is pushed to the `main` branch, the LaTeX source code is automatically compiled into a PDF. 

**To download the latest compiled report:**
1. Go to the [Actions tab](../../actions).
2. Click on the latest successful build (green checkmark).
3. Scroll down to **Artifacts** and download `Final-Report-PDF`.

## 🛠️ Tech Stack & Tools
* **Document Preparation:** LaTeX
* **CI/CD:** GitHub Actions
* **Core Technologies Discussed:** Cloud Computing, K3s, Kubernetes, Edge Nodes, IoT
